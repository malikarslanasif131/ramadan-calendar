# 🌙 Ramadan Calendar 1938–2038

> A beautifully designed, fully interactive 100-year Ramadan calendar — built with pure HTML, CSS & JavaScript. No frameworks, no dependencies, no backend.

## ✨ Features

- 📅 **100 Years of Data** — Every Ramadan start date from 1938 to 2038 (Hijri 1357–1460)
- 🗂️ **Grouped by Decade → Month** — Clear visual hierarchy for easy browsing
- ⏳ **Live Countdown** — Shows days until / days since each Ramadan
- 🌙 **Next Ramadan Highlight** — Auto-detects and highlights the upcoming Ramadan
- 🔍 **Smart Filters** — Search by year, month, season, day of week, or decade
- ⊞ **Grid & Table Views** — Switch between card grid and compact table layout
- 📱 **Fully Mobile Responsive** — Optimized for phones, tablets, and desktops
- 🎨 **Premium Dark UI** — Star field background, gold accents, animated moon, glassmorphism cards
- ✦ **Season Tags** — Each date tagged with ❄️ Winter / 🌸 Spring / ☀️ Summer / 🍂 Autumn
- 🕌 **Arabic Typography** — Amiri serif font for authentic Islamic aesthetic

## 🚀 Getting Started

### Option 1 — Open Directly
Just download and open `ramadan-calendar.html` in any browser. No server needed.

```bash
git clone https://github.com/malikarslanasif131/ramadan-calendar.git
cd ramadan-calendar
# open ramadan-calendar.html in your browser
```

### Option 2 — GitHub Pages
Push to your repo and enable GitHub Pages to get a live URL instantly.

1. Go to **Settings → Pages**
2. Set source to `main` branch, `/ (root)`
3. Visit `https://malikarslanasif131.github.io/ramadan-calendar`

## 🔍 Filter Guide

| Filter | What it does |
|--------|-------------|
| **Search** | Match any year (e.g. `2025`), month name (e.g. `March`), or Hijri year |
| **Decade** | Show only a specific 10-year period |
| **Season** | Filter by ❄️ Winter, 🌸 Spring, ☀️ Summer, 🍂 Autumn |
| **Month** | Show only a specific Gregorian month |
| **Day of Week** | Find all Ramadans that started on a Friday ⭐ |
| **Reset** | Clear all filters and show all 100 years |

## 📱 Responsive Breakpoints

| Screen | Behavior |
|--------|----------|
| **Desktop** (> 768px) | Full grid layout, all filters visible, horizontal card rows |
| **Tablet** (≤ 768px) | Filters stack vertically, cards scroll horizontally per month |
| **Mobile** (≤ 480px) | Compact cards (145px), iOS zoom prevention, hidden minor columns |
| **Small** (≤ 360px) | Minimal card size, table row numbers hidden |
